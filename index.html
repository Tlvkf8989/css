선언방식

1)내장방식

style태그를 이용해 css를 작성하는 방식
위의 방식을 사용하면 별도의 css파일을 만들지 않고도 HTML에서 바로 스타일 적용 가능

2)링크방식

CSS파일을 HTML의 Link태크로 연결하는 방법
연결된 파일을 한번에 가져오므로 파일을 불러오는 속도가 비교적 빠름

3)인라인방식

HTML의 전역 속성으로 스타일을 직접 추가

4)Imput방식

ID선택자

HTML ID속성의 값으로 요소를 선택.
ID선택자에는 #기호가 사용된다
예를들어 css에서 #orange 선택자는 HTML 파일에서 id="orange"라는 속성과 값을 가진 요소를 선택함
